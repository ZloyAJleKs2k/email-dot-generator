# Email Dot Generator

Генератор email-адресов с добавлением точек в локальную часть (до @), основанный на исходном email.

## Описание

Программа генерирует все возможные варианты email-адресов, вставляя точки в различные позиции локальной части исходного email. Например, из `test@gmail.com` можно получить `t.est@gmail.com`, `te.st@gmail.com`, `t.e.s.t@gmail.com` и т.д.

## Функциональность

- Генерация всех возможных комбинаций точек в локальной части email

## Требования

- Python 3.x
- Библиотека `itertools` (входит в стандартную библиотеку Python)

## Использование

1. Установите свои параметры в блоке `if __name__ == "__main__":`:

```python
original_email = "yourgmail@gmail.com"  # Замените на свой email
output_file = "emails_with_dots_full.txt"  # Путь к выходному файлу
max_count = 200  # Максимальное количество генерируемых email
```


## Параметры

- `email`: Исходный email-адрес для генерации
- `output_file`: Файл для сохранения результата
- `max_emails`: Максимальное количество генерируемых email-адресов (по умолчанию 1000)

## Пример

Для email `test@gmail.com` будут сгенерированы:
```
test@gmail.com
t.est@gmail.com
te.st@gmail.com
t.e.st@gmail.com
...
```
